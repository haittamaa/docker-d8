Pré-requis : Docker & Docker compose
Root folder : mywebsite    

Commandes utiles :

    Lancer docker     : sudo docker-compose up -d

    Arreter docker    : sudo docker-compose down

    Accès SSH & Drush : sudo docker exec -it mywebsite_web_1 bash

    Accès web : website.local:8080

    Config SQL :

    host : mywebsite_1
    user : root
    pass : mysqlroot
